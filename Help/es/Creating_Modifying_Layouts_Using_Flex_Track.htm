Usar vías flexibles (Flex Track)


Las vías flexibles (Flex Track) están junto con el resto de tipos de vía LEGO. También puedes usar el atajo de teclado 'F' para añadir un tramo de vías flexible. La principal diferencia es cómo se puede "flexionar" la vía para seguir una forma después de que varias piezas individuales se hayan conectado.

Para flexionar una sección de vías (un tramo flexionable) se debe seleccionar el conjunto de piezas que se quieran flexionar, haciendo entonces doble clic sobre la vía que se quiere mantener sujeta, manteniendo pulsado el botón izquierdo del ratón en el segundo clic. Si se hace correctamente, el icono del ratón cambiará de un ladrillo a una flecha con el símbolo de vía flexible. Mientras se mantenga el botón del ratón pulsado, la vía puede flexionarse como una serpiente en los mismos rangos que si fuera real, moviendo el ratón en la dirección de giro deseada. Para dejar de flexionar la vía, simplemente hay que soltar el botón izquierdo del ratón.

Un tramo de vías flexionable puede contener también rectas o curvas, no necesita contener sólo vías flexibles, con lo que se puede controlar cuáles piezas deben estar incluidas en el tramo flexionable. Además, el tramo flexionable se forma empezando desde la pieza sobre la que hagas doble clic y siguiendo los enlaces de conexión tanto como las piezas estén conectadas y tengan sólo dos conexiones. Cuando se alcanza una pieza con más de dos conexiones (como un cruce, etc.) la elección del tramo flexionable se detiene.

El movimiento del tramo flexionable sigue un algoritmo convergente iterativo que exige cierto cálculo matemático. Si quieres conectar un tramo flexionable a una vía existene, a veces es necesario esperar unas décimas de segundo para dejar al algoritmo encontrar la solución óptima.
