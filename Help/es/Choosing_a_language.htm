Choosing a language

The first thing you may wish to do is change the software language. In order to do this you go to  edit > global options > general  and chose the language you need from the drop down list in the top left corner. Press ok to close the window and restart the program for the changes to take effect.

If this help file was translated in your language, next time you choose  help > content and index , the help will be displayed in the language chosen for the application.
