Áreas de la ventana

La ventana de BlueBrick tiene 3 áreas: el área de trazado, el área de piezas y el área de capas. El uso de estas áreas se explica con más detalle en la sección Crear y modificar modelos.

El área de trazado

El área de trazado ocupa la parte izquierda de la pantalla y se reconoce por su fondo azul con cuadrícula blanca (por defecto). Este área tiene una herramienta modificable de "ajuste a la cuadrícula" que te ayudará a ajustar diferentes elementos en tu trazado.

El área de piezas

El área de piezas se encuentra en la esquina superior derecha de la ventana principal y consiste en varias pestañas de categorías en las que las piezas están organizadas. Si deseas contribuir a la lista de piezas por favor pulsa aquí. En la actualidad la pestaña "Custom" contiene 3 diferentes tamaños de mesas. Los tamaños representan el número de baseplates de 32x32 que caben en la superficie.

La área de capas

El área de capas se encuentra en la esquina inferior derecha. Esta herramienta es especialmente práctica para trazado complejos. En este área puedes añadir, eliminar y organizar las diferentes capas que necesites. Se pueden encontrar más detalles en la sección Crear y usar capas.
