Crear y usar capas

Las capas añaden mayor flexibilidad y facilitan la organización del un trazado. Aquellos que hayan utilizado programas de edición de fotografías como Photoshop estarán familiarizados con este sistema. Pero en BlueBrick hay distintos tipos de capas para manejar distintos tipos de objetos como textos, reglas y, por supuesto, piezas. Esto también significa que sólo puedes crear estos objetos en una capa del tipo correspondiente.

En esta parte puedes aprender cómo usar la barra de herramientas de capas y la diferencia entre los cinco diferentes tipos de capas.
