Exportar una lista de piezas


Exportar

Exportar una lista de piezas es una importante función cuando necesitas una lista de comprobación o una lista de compras. Esta función se encuentra bajo  ver > lista de piezas...  .

Puedes separar la lista por capas marcado la caja al final de la ventana.

Para guardar la lista pulsa el botón "Exportar". Puedes guardar la lista como un archivo de texto o como un archivo HTML si quieres incluir imágenes de las piezas en la lista. Las imágenes de las piezas enlazan por internet, lo que hace la lista fácil de transportar (la puedes enviar a un amigo), aunque por supuesto requiere conexión a internet para visionarla correctamente.

Imprimir

La lista se puede imprimir abriendo el archivo de texto o HTML que exportaste usando la función de exportación, y usando el comando de impresión del programa con que abres el archivo.
