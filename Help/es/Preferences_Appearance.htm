Preferencias de apariencia

El esquema de colores 3 por defecto es "Blue Print". También puedes seleccionar las versiones Pergamino o Clásico. Puedes crear esquemas de colores personalizados seleccionando la opción "Personalizado de la lista desplegable y seleccionando tus colores en las cajas de colores al lado de las descripciones de texto : "Color de fondo", "Color de cuadrícula " y "Color de sub-cuadrícula"

You can also adjust the highlight percentage for the selected parts and for the single part you grab in a group of selected parts.
El resultado de tu selección de colores se mostrará en la ventana de muestra. Los cambios no tendrán efecto hasta que pulses "OK".

You can also set the default text format 4 (font style and color) which will be used each time you create a new text on your map.

Similarly you can set the default settings for each new grid layer 5 and each new area layer 6. To tune the default ruler style see the Edition Preference section.
