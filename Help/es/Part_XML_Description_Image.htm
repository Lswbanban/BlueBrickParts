Especificar una imagen de referencia en Internet

Sintaxis: <ImageURL>...</ImageURL>
Valor por defecto: Por defecto BluBrick utiliza la siguiente URL:
http://media.peeron.com/ldraw/images/<ColorCode>/<PartNumber>.png
Descripción: Se puede usar una dirección de Internet completa (URL) que lleva a una imagen de la pieza, como este URL, para generar una lista de piezas en HTML.

Cuando exportas una lista de piezas de tu trazado en HTML, para hacer un archivo transportable, las imágenes de las piezas en la lista no van enlazadas con una ubicación en tu disco duro. En su lugar empleamos una URL de la imagen, como un enlace a Bricklink o Peeron. Si has creado una pieza personal, puedes subir una imagen a tu cuenta en Brickshelf, y especificar la URL del enlace directo en la etiqueta XML.
