Specify an online reference Image

Syntax: <ImageURL>...</ImageURL>
Default Value: By default BlueBrick construct this URL:
http://media.peeron.com/ldraw/images/<ColorCode>/<PartNumber>.png
Description: A complete internet address (URL) that point to a picture of the part, such as this URL can be used to generate the Part List in HTML.

When you export the part list of your map in HTML, in order to have a portable file, the images of the parts in the list do not reference your local hard drive. Instead we use an internet URL such as a link to the Peeron or Bricklink websites. If you have created a custom part, you can upload a picture of it to your Brickshelf folder, and specify the Brickshelf deep link URL in the XML tag.
