Especificar autor

Sintaxis: <Author>...</Author>
Valor por defecto: Texto "Unknown".
Descripción: El nombre del autor de esta pieza.

Este campo puede contener texto libre para dar información sobre el autor de la pieza. Puede contener tanta información como deseas, como un correo electrónico por ejemplo. Esta información es ignorada por BlueBrick durante el proceso de revisión, pero será usada en el futuro.
