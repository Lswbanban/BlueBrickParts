Especificar si el grupo se puede desagrupar

Sintaxis: <CanUngroup>...</CanUngroup>
Valor por defecto: true (verdadero)
Descripción: Una etiqueta booleana "true" o "false" (sensible a mayúsculas) que especifica si un grupo se puede desagrupar o no en BlueBrick.

Por defecto cualquier grupo puede ser deagrupado después de añadirlo a un trazado (vea la documentación sobre agrupar/desagrupar piezas). Si esta etiqueta tiene el valor "false", el usuario no podrá desagruparlo haciendo un clic derecho sobre el trazado.

Esta etiqueta tiene bastante importancia para asegurar una buena experiencia de uso. Dependiendo de qué tipo de grupo creas, puedes querer permitir que un usuario o desagrupe o no. Por ejemplo, si estás haciendo un atajo para introducir un módulo 3x3 estándar, querrás dejar que otros usuarios lo modifiquen luego de añadirlo a su trazado. Por otro lado, si creas un grupo para un set que se puede abrir con un hinge, no querras permitir que el usuario desarme el grupo por accidente.
