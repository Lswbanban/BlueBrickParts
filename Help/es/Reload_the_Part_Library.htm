Recargar la biblioteca de piezas

Los usuarios que crearon nuevas piezas para BlueBrick solicitaron una opción muy práctica. Esta opción es la posibilidad de recargar la biblioteca de piezas sin cerrar y reiniciar el programa. Esta poción está disponible en  Archivo > Recargar biblioteca de piezas. Como el programa puede tardar varios segundos en cargar, esto sería una pérdida de tiempo al intentar ajustar una nueva pieza en muchas iteraciones.

El proceso correcto para usar esta opción es primero hacer clic en  Archivo > Recargar biblioteca de piezas . Si no guardaste el trazado aparecerá un mensaje de aviso porque el mapa se descargará. Luego se descarga también la biblioteca de piezas y aparece un mensaje invitándote a recargar la biblioteca cuando estés listo.

Normalmente no aceptas enseguida, sino que cambias a otro programa como un editor de imagen o de XML. Cuando hayas terminado de modificar la pieza, puedes pulsar en el botón ok del mensaje y se recargará la biblioteca de piezas, así como el trazado que anteriormente estaba abierto.
