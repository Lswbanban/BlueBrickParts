Especificar si la pieza debe ser escondida.

Sintaxis: <NotListedInLibrary>...</NotListedInLibrary>
Valor por defecto: false
Descripción: Si este valor es "true", la pieza no será visible en la biblioteca de piezas del programa.

La mayoría de piezas deberían ignorar esta etiqueta. Si este valor es "true", la pieza no será visible en la biblioteca de piezas del programa. Si el valor es "false" la pieza aparecerá de forma normal en la biblioteca de piezas de BlueBrick. Esta etiqueta es práctica si quieres crear un grupo que sea accesible en la biblioteca de piezas pero no quieres que aparezcan los elementos individuales.

Incluso si esta etiqueta tiene el valor "true", BlueBrick cargará la imagen asociada (si la hay), y todas las propiedades XML además de hacer la pieza completamente funcional en el mapa (puntos de conexión, etc...). En otras palabras, si esa pieza escondida aparece en un grupo en tu trazado, desagrupa el grupo y podrás manipular la pieza como cualquier otra. Asignar el valor "true" simplemente evitará que la pieza aparezca en la biblioteca para poder ser arrastrada al mapa.
