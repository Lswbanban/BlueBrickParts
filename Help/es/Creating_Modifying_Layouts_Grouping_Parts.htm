Agrupar Piezas


La opción de Agrupar se usa para crear un grupo a partir de varias piezas. Esta función se usa principalmente para manipular las piezas con más facilidad, ya que puede reducir el número de piezas individuales que necesitan ser seleccionadas.

También puede ser útil si tu diseño utiliza con frecuencia una combinación idéntica de piezas con la misma configuración. Esto puede incluir tramos de 1/4 o 1/2 de curva, o un patrón repetitivo de carreteras o otras baseplates. Paara copiar el grupo sólo tendrás que hacer clic en una pieza cualquiera que forme parte de él, en lugar de en todas las piezas de forma individual.

Para agrupar una combinación de piezas, en primer lugar tienes que seleccionar todas las piezas que quieres tener en el grupo, entonces hacer clic derecho sobre una de las piezas seleccionadas y elegir en el menú desplegable "Agrupar". Para desagrupar, primero hay que seleccionar una de las piezas del grupo con el botón derecho del ratón y, entonces, elegir "Desagrupar".

También puedes tener grupos dentro de un grupo. Para ello, selecciona un grupo existente (lo llamaremos grupo 1) junto con varias piezas sueltas que quieras incluir en el nuevo grupo (grupo 2), y entonces selecciona la función "agrupar" tal como se indicó más arriba. Si se seleccionara cualquiera de las partes del grupo 1 y se eligiera "desagrupar", desagruparía el "grupo 2", dejando el "grupo 1" intacto. Si lo que se quiere es desagrupar el grupo 1, se deberá aplicar la función una vez más.

La función de agrupado funciona exáctamente igual con piezas, textos y reglas. Además, esta función se puede deshacer y rehacer, consulta Deshacer y rehacer para más detalles..

Se puede guardar un grupo en la "Biblioteca de piezas" mediante "Archivo > Grabar Selección en la bilioteca... para facilitar el trabajo posterior. Para más información, puedes leer la sección "Cómo guardar un grupo en la Biblioteca de piezas"
