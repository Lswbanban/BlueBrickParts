Los cuatro tipos de capas


Capas de cuadrícula

Por defecto, un nuevo trazado siempre comienza con una capa de cuadrícula con una cuadrícula en blanco 11. Si no quieres que un trazado contenga una capa de cuadrícula por defecto, puedes modificar esto en las opciones generales. Puedes añadir más capas de cuadrícula pulsando 3. Una capa de cuadrícula adicional puede ser práctica si quieres tener coordenadas verticales y horizontales en capas separadas para poder esconder una u otra.

Estas capas siempre son transparentes y contienen coordenadas de cuadrícula. Puedes acceder a las opciones de cuadrícula y texto haciendo doble clic sobre la pestaña de la capa de cuadrícula 10. Puedes cambiar el color de fondo mediante el menú  editar > fondo de mapa . Vea la sección Opciones de apariencia para más detalles.

Ten en cuenta que haciendo clic sobre la cuadrícula y arrastrándola, moverás las coordenadas iniciales de los ejes X e Y.

Capas de ladrillos

Las capas de ladrillos se usan para insertar los elementos LEGO®. Tener varias capas puede facilitar la tarea de editar y organizar un trazado complejo. En la captura de pantalla de arriba se usaron tres capas de ladrillo diferentes: Una para la mesa 13, una para las carreteras y edificios y otra para las vías. Esta opción también es práctica si tienes trazados que combinan diferentes tipos de vías. Por ejemplo, puedes usar una capa para 9V, una para 12V y otra para monorail. El contenido de cada capa se puede mover independientemente de las demás capas.

Por defecto, un nuevo diseño también tiene inicialmente una capa de piezas que está seleccionada al crearlo, por lo que se puede empezar a construir directamente tras arrancar Bluebrick, o al crear un nuevo diseño al seleccionar el menú  Archivo > Nuevo  . Pero como en el caso de la cuadrícula por defecto, se puede cambiar este comportamiento en las Preferencias generales. Para añadir una nueva capa de piezas, pulsa el botón 4.

Capas de área

Las capas de área se usan para marcar o reservar secciones en y alrededor del diseño que posiblemente no se necesiten para LEGO, o una parte de tu trazado fijo 12. En la captura de pantalla de arriba, el área marcado indica la ubicación del controlador 9V. Estas secciones se marcan con la herramienta de pintar que está activa sólo cuando se selecciona una capa de área. Para añadir un nuevo área, pulsa el botón 5. Puedes cambiar el color simplemente haciendo clic sobre el botón de la herramienta de pintar 15, o puedes seleccionar el color deseado a través de  editar > herramienta de pintar... > elegir color  menu. Haciendo doble clic sobre la pestaña de la capa de área te dará acceso a las opciones de transparencia y el tamaño del pincel en studs.

Todo el área se puede mover pulsando la tecla CTRL (por defecto) y arrastrándola a su nueva ubicación con el ratón. Nota que se moverá todo el contenido de la capa.

Las áreas pintadas pueden ser borradas haciendo clic sobre flecha al lado de la herramienta de pintar 15 y seleccionando la herramienta de borrar. Desafortunadamente, no hay forma de copiar/pegar o mover sólo una parte del área.

Capas de texto

La capas de texto son prácticas para marcar secciones del trazado 14. Pulsa 6 y luego haz doble clic dentro del área de la capa para abrir una caja de texto. Escribe el texto y modifica las propiedades del texto a voluntad. Pulsa OK para cerrar la ventana y luego coloca el texto en la posición deseada con el ratón. El texto también se puede rotar, usando los botones para rotar y los ajustes de rotación. For more details about Text editing, please read the Using Texts section.

Al igual que con cualquier tipo de capa, también se puede modificar la transparencia de una capa de texto haciendo doble clic en la pestaña de la capa 10. Así, si quieres tener diferentes niveles de transparencia para diferentes textos, podrás hacerlo usando varias capas de texto.

Capas de Reglas

Las capas de reglas son útile spara medir distancias en tu diseño. Puedes añadir una nueva capa de reglas pulsando el botón 7. En una capa de reglas puedes añadir reglas lineales y circulares que pueden mostrar la distancia en varias unidades (Stud, LDU, tramos de recta, módulos, metros y pies). Estas reglas también pueden usarse para, símplemente, delimitar los muros de una sala de exposición donde instalarás tu diseño. Las reglas también se pueden fijar a piezas situadas bajo las mismas, con lo que la regla actualizará automáticamente sus dimensiones cuando se mueva la pieza. Para más detalles sobre las capas de reglas, por favor consulta la sección Usar Reglas.
