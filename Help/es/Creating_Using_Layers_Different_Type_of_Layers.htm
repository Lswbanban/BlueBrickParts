Los cuatro tipos de capas


Capas de cuadrícula

Por defecto, un nuevo trazado siempre comienza con una capa de cuadrícula con una cuadrícula blanca 11. Si no quieres que un trazado contenga una capa de cuadrícula por defecto, puedes modificar esto en las opciones generales. Puedes añadir más capas de cuadrícula pulsando 3. Una capa de cuadrícula adicional puede ser práctica si quieres tener coordenadas verticales y horizontales en capas separadas para poder esconder una u otra.

Estas capas siempre son transparentes y contienen coordenadas de cuadrícula. Puedes acceder las opciones de cuadrícula y texto haciendo doble clic sobre la pestaña de la capa de cuadrícula 10. Puedes cambiar el color de fondo mediante el menú  editar > fondo de mapa . Vea la sección Opciones de apariencia para más detalles.

Nota que haciendo clic sobre la cuadrícula y arrastrándola, moverás las coordenadas iniciales de los ejes X e Y.

Capas de ladrillos

Las capas de ladrillos se usan para insertar los elementos de LEGO®. Tener varias capas puede facilitar la tarea de editar y organizar un trazado complejo En el pantallazo de arriba se usaron tres capas de ladrillo diferentes. Una para la mesa 13, una para las carreteras y edificios y otra para las vías. Esta opción también es práctica si tienes trazados que combinan diferentes tipos de vías. Por ejemplo, puedes usar una capa para 9V, una para 12V y otra para monorail. El contenido de cada capa se puede mover independientemente de las demás capas.

By default, a new layout also starts with an initial Brick Layer and this layer is selected by default, so you can directly start building your layout after starting BlueBrick, or after starting a new layout via the  File > New  menu. But, as for the default Grid Layer, you can change this behavior in the General Preferences. To add a new Brick Layer, press the button 4.

Capas de área

Las capas de área se usan para marcar o reservar secciones en y alrededor del trazado que posiblemente no se necesiten para LEGO, o una parte de tu trazado fijo 12. En el pantallazo arriba, el área marcado indica la ubicación del controlador 9V. Estas secciones se marcan con la herramienta de pintar which is only active when you select an Area Layer. To add a new Area Layer, press the button 5. Puedes cambiar el color simplemente haciendo clic sobre el botón de la herramienta de pintar 15, o puedes seleccionar el color deseado a través de  editar > herramienta de pintar... > elegir color  menu. Haciendo doble clic sobre la pestaña de la capa de área te dará acceso a las opciones de transparencia y el tamaño del pincel en studs.

Todo el área se puede mover pulsando la tecla CTRL (por defecto) y arrastrándola a su nueva ubicación con el ratón. Nota que se moverá todo el contenido de la capa.

Las áreas pintadas pueden ser borradas haciendo clic sobre flecha al lado de la herramienta de pintar 15 y seleccionando la herramienta de borrar. Unfortunately there's no way for now to copy/paste or move only a part of the area.

Capas de texto

La capas de texto son prácticas para marcar secciones del trazado 14. Pulsa 6 y luego haz doble clic dentro del área de la capa para abrir una caja de texto. Escribe el texto y modifica las propiedades del texto a voluntad. Pulsa OK para cerrar la ventana y luego coloca el texto en la posición deseada con el ratón. El texto también se puede rotar, usando los botones para rotar y los ajustes de rotación. For more details about Text editing, please read the Using Texts section.

As for every type of layer, you can also change the transparency of the Text Layer by double clicking on the text layer tab 10. So if you want to have different transparencies for different texts, just use different Text Layers.

Ruler Layers

Ruler Layers are useful to measure distances on your layout. You can add a new Ruler Layer by pressing the button 7. In a Ruler Layer you can add linear and circular rulers which can display a distance in various unit (Stud, LDU, Straight Track, Module, Meter, Feet). These rulers can also be used to simply draw lines and circles on your layout, for example to delimit the wall of the room where you plan to build your layout. Rulers can also be attached to parts underneath such as the ruler will automatically extends and updates its measured value when you move the part. For more details about Ruler editing, please read the Using Rulers section.
