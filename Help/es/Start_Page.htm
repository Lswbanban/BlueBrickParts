BlueBrick
 Versión 1.6.1

 por Alban Nanty

BlueBrick es un programa libre para Windows/Vista desarrollado por Alban Nanty en open source Puedes usarlo para tus proyectos personales, pero no para uso comercial. Este programa se distribuye con la esperanza de que sea de uso, pero SIN NINGUNA GARANTÏA.

Este programa se diseñó especialmente para los AFOLs que quieren preparar trazados para exposiciones con LEGO® BlueBrick es compatible con LDRAW y Train Depot Track Designer y se diseñó para extender su base de piezas de manera sencilla. Sus capas permiten una mejor organización de tu mapa, y algunas capas específicas permiten añadir anotaciones y asignación de áreas.
