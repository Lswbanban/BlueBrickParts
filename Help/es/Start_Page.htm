BlueBrick
 Versión 1.6.1

 por Alban Nanty

BlueBrick es un programa gratuíto para Windows desarrollado por Alban Nanty como código abierto. Puedes usarlo para tus proyectos personales, pero no para uso comercial. Este programa se distribuye con la esperanza de que sea de utilidad, pero SIN NINGUNA GARANTíA.

Este programa se diseñó especialmente para los AFOLs que quieren preparar trazados para exposiciones LEGO®. BlueBrick es compatible con LDRAW y Train Depot Track Designer y se diseñó para extender su base de piezas de manera sencilla. Su trabajo por capas permite una mejor organización de tu diseño, y algunas capas específicas permiten añadir anotaciones y asignación de áreas.
