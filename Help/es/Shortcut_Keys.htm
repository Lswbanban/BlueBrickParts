Atajos de teclado


Se pueden añadir atajos de teclado utilizando el menú desplegable 4 al final de la lista de atajos de teclado (fig. 9c) Elige las teclas del teclado a las que deseas asignar una función 1, luego la función que esta tecla debe cumplir 2, la pieza o el número de set que usará 3, y el tipo de conexión que tendrá 4. Parta completar el proceso pulsa el botón "Añadir atajo de teclado" 5.

Nota que un atajo de teclado no se puede editar una vez creado. Debes borrar el atajo si deseas cambiarlo y crear uno nuevo. Pulsar el botón "Restaurar opciones por defecto" 6 se borrará cualquier atajo personalizado que hayas creado. Estos cambios serán permanentes una vez que pulses el botón "OK". Si pulsas "Cancelar" los cambias se descartarán.

Nota que el botón "Restaurar opciones por defecto" 6 resetea todas las opciones de las 3 pestañas a la vez. No solo resetea las configuraciones de la pestaña que estás viendo. Estos cambios serán permanentes una vez que pulses el botón "OK". Si pulsas "Cancelar" los cambias se descartarán.
