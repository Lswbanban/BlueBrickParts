Ajouter des Pièces à la Bibliothèque

Un des points fort de BlueBrick est la facilite avec laquelle des pièces peut être ajouter à la bibliothèque. La bibliothèque de pièces est créé quand l'application est démarrée, en utilisant les fichiers présents sur votre disque dur. Ajouter ou enlever des pièces est fait en ajouter ou effaçant des fichiers dans le répertoire de pièces. Si vous voulez connaitre comment créer et ajouter vos propres pièces à BlueBrick, vous devriez suivre ces étapes:

  1. Premièrement regarder commentle répertoire de pièces est organise.
  2. Puis créé une nouvelle image pour votre pièce.
  3. Éventuellement, vous pouvez ajouter une description de piècesdans un fichier xml.
  4. Vous devriez lire la section "Réinitialisation de la Bibliothèque de Pièces". Ceci est très utile durant le développement de nouvelle pièces.
  5. Enfin, vous pouvez aussi lire la documentation sur le fichier de configuration des "Types de Connection" si vous voulez ajouter vos propre types de connection.
