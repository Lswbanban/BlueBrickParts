Non affiché dans la bibliothèque

Syntaxe : <NotListedInLibrary>...</NotListedInLibrary>
Valeur par défaut : false
Description : Si ce label est positioné à "true", la pièce ne sera pas visible dans la bibliothèque de pièces de l'application.

La plupart des pièces ne devraient pas avoir besoin de ce label. Si ce label est positionné à "true" (vrai), la pièce ne sera pas visible dans la bibliothèque des pièces. Si le label est absent ou positionné à "false" (faux), la pièce apparaitra alors normalement dans la bibliothèque. Ce label est utile lorsque vous souhaitez créer un groupe accessible dans la bibliothèque pièces mais que vous voulez masquer toutes les sous-pièces du groupe.

A noter que même si cet indicateur est positionné à true, BlueBrick chargera les images associées (si existantes) ainsi que les propriétés XML et la pièce sera totallement fonctionnelle (points de connexion, etc.) sur la carte. En particulier, si vous ajoutez cette pièce via un groupe et ensuite dégroupez le groupe, vous pourrez manipuler la pièce normallement, comme n'importe quelle autre. Positionner le label à true bloque uniquement l'affichage de la pièce dans la bibliothèque ainsi que la possibiliter de la placer sur la carte.
