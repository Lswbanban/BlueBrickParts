Spécifier l'auteur

Syntaxe: <Author>...</Author>
Valeur par défaut: Le texte "Inconnu".
Description: Le nom de l'auteur de l'élément.

Ce champ peut contenir du texte afin d'indiquer l'auteur de l'élément. Il peut contenir autant d'informations que vous le voulez, comme votre adresse e-mail par exemple. Cette information est ignorée par BlueBrick durant l'analyse, mais va être utilisée dans le futur
