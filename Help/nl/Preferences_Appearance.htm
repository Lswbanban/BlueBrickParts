Beschikbare verbindingspunten

Het standaard kleurenschema 3 is "Blue Print". Je kunt ook voor klassieke versies kiezen. Aangepaste kleurenschema's kunnen worden gemaakt door te kiezen voor "Custom" uit drop-down lijst en het kiezen van de gewenste kleuren door te klikken op de kleur vakjes naast de tekst vermeldingen, "Achtergrondkleur", "Standaard RasterKleur" en "Standaard sub-raster kleur".

You can also adjust the highlight percentage for the selected parts and for the single part you grab in a group of selected parts.
Het resultaat van de kleur selectie zal worden weergegeven in het voorbeeldvenster. De veranderingen zullen pas van kracht na het klikken op de "OK" button.

You can also set the default text format 4 (font style and color) which will be used each time you create a new text on your map.

Similarly you can set the default settings for each new grid layer 5 and each new area layer 6. To tune the default ruler style see the Edition Preference section.
