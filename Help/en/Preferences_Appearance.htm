Appearance Preferences

The default Color Scheme 3 is "Blue Print" You can also select a Parchment or Classic version. Custom color schemes can be made by choosing your colors by clicking on the color boxes next to the following text entries: "Background Color", "Default Grid Color" and "Default sub-grid Color". You can also adjust the highlight percentage for the selected parts and for the single part you grab in a group of selected parts. The result of your color settings will be shown in the preview window. The changes will not take effect until after press the "Ok" button.

You can also set the default text format 4 (font style and color) which will be used each time you create a new text on your map.

Similarly you can set the default settings for each new grid layer 5 and each new area layer 6. To tune the default ruler style see the Edition Preference section.
