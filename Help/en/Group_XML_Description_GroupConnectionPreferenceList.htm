List the interface connections of the group

Syntax: <GroupConnectionPreferenceList>
    <nextIndex from="...">...</nextIndex>
    ...
</GroupConnectionPreferenceList>
Default Value: If this tag is not present, the group cannot be connected using its connection points.
Description: A list of connections.

The GroupConnectionPreferenceList field is required if your group contains parts that have connection points and you want to be able to connect the whole group to an existing part via its connection points. This field contains a list of nextIndex field, each entry indicating a connection point that can be used to connect the group. When grouping several parts that have connection points, the sum of all these connection points can potentially become the connection points of the group. However, you may not use all of them.
