Creating and Using Layers

Layers add extra flexibility and make your layout easier to organize. Users of photo editing packages such as Photoshop will be familiar with this system. But in BlueBrick there are different types of Layers which handle different types of objects such as Texts, Rulers and of course Parts. This also means that you can only create those objects on a layer of their corresponding type.

This is why you should read in this section how to use the Layer Toolbar and the difference between the five different types of layers before starting to create your layout.
