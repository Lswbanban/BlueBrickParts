How to customize the Shortcut Keys

Hinzufügen von Tastaturkürzeln ist mit Hilfe der vier Auswahlmenüs unterhalb der Tastaturkürzel-Liste (Menü Bearbeiten, Allgemeine Einstellungen...) möglich:

Wähle zuerst die Taste aus 3, der eine Funktion zugeordnet werden soll, und dann die Funktion 4. Bei der Funktion "Teil hinzufügen" muss noch die Teil- oder Setnummer ausgewählt 5 und der Verbindungstyp 6 angegeben werden. Zum Abschluss muss nur noch die Schaltfläche "Tastenkürzel hinzufügen" 7 angeklickt werden.

Ein Tastaturkürzel kann nicht direkt geändert werden. Um einer Taste eine andere Funktion zuzuweisen, muss zunächst das Tastaturkürzel gelöscht werden. Danach kann der Taste eine neue Funktion zugewiesen werden.

Mit Anklicken der Schaltfläche "Standard wiederherstellen" 2 werden alle selbst erstellten Tastaturkürzel gelöscht. Pressing the "Restore All to Default" button 1 will also delete any change in the shortcuts but will also resets the settings on all the tabs at the same time.

Wird dann die Schaltfläche "Ja" angeklickt, werden die Standardeinstellungen wieder eingestellt.
