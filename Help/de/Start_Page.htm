BlueBrick
 Version 1.9.2

 by Alban Nanty

BlueBrick ist eine von Alban Nanty entwickelte, freie Open-Source-Software für Windows/Vista. Für dieses Programm wird keinerlei Garantie oder Gewährleistung übernommen.

Diese Software wurde besonders für AFOLs zur Vorbereitung von LEGO®-Ausstellungen entwickelt. BlueBrick ist kompatibel mit LDRAW, Train Depot Track Designer und 4DBrix nControl. BlueBrick wurde so gestaltet, dass die Bibliothek der in BlueBrick verwendeten Teile einfach erweitert werden kann. Die Ebenen-Funktion erlaubt die übersichtliche Gliederung von Gleisplänen, in speziellen Ebenen können Kommentare eingefügt und besondere Flächen gekennzeichnet werden.
