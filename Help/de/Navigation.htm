Navigation

BlueBrick wurde so gestaltet, dass die Navigation in der Gleisplan-Ansicht (siehe auch Bildschirmaufteilung) so einfach wie möglich ist. Dies trifft besonders dann zu, wenn eine Maus mit einem dritten Mausbutton in Form eines Rades benutzt wird. Selbstverständlich kann BlueBrick auch auf einem Laptop mit Touchpad genutzt werden.

In diesem Abschnitt wird gezeigt, wie man eine Ansicht zoomt und verschiebt und wie man Teile auswählt.
