Die vier verschiedenen Arten von Ebenen


Rasterebenen

Aufgrund der Standardeinstellungen beginnt jeder neuer Gleisplan mit einer Rasterebene mit einem weißen Raster 11. Soll ein neuer Gleisplan nicht mehr automatisch eine Rasterebene enthalten, so muss die entsprechende Standardeinstellung, wie im Abschnitt Allgemeine Einstellungen beschrieben, geändert werden. Durch Anklicken der Schaltfläche 3 können weitere Rasterebenen hinzugefügt werden. Eine zusätzliche Rasterebene ist sinnvoll, wenn horizontale und vertikale Koordinaten getrennt voneinander angezeigt werden sollen.

Rasterebenen sind immer transparent und enthalten immer Rasterkoordinaten. Mit einem Doppelklick auf die Rasterebenenfläche 10 gelangt man zu den Raster- und Texteinstellungen. So kann z.B. die Hintergrundfarbe mit dem Menü  Bearbeiten > Hintergrund  geändert werden. Weitere Details werden im Abschnitt Aussehen (Erscheinungsbild) einstellen erläutert.

Durch Einrichtung einer Ebene nur mit vertikalen Koordinaten und einer Ebene nur mit horizontalen Koordinaten lassen sich durch Ausblendung der jeweils anderen Ebene nur die horizontalen bzw. vertikalen Koordinaten anzeigen.

Steineebenen

In Steineenbenen werden LEGO®elemente plaziert und damit ein Gleisplan erstellt. Mit mehreren Steinebenen lassen sich auch komplexe Gleispläne übersichtlich erstellen und bearbeiten. Im oben dargestellten Screenshot wurden 3 verschiedene Steinebenen benutzt: eine für Tische 13, eine für Straßen und Gebäude und eine für Gleise. Verschiedene Steinebenen sind auch hilfreich, wenn unterschiedliche Gleistypen (9v, 12V, Monorail) in einem Gleisplan verwendet werden. Der Inhalt einer Ebene kann unabhängig von den anderen Ebenen verschoben und geändert werden.

By default, a new layout also starts with an initial Brick Layer and this layer is selected by default, so you can directly start building your layout after starting BlueBrick, or after starting a new layout via the  File > New  menu. But, as for the default Grid Layer, you can change this behavior in the General Preferences. To add a new Brick Layer, press the button 4.

Flächenebenen

Flächenebenen sind nützlich, um Abschnitte des Gleisplans zu markieren oder zu reservieren, die nicht unbedingt für Legoteile benötigt werden 12. Im oben gezeigten Screenshot wurde eine Flächenebenen genutzt, um den Bereich für den 9V-Trafo zu kennzeichnen. Derartige Abschnitte können mit dem Zeichnen-Werkzeug markiert werden which is only active when you select an Area Layer. To add a new Area Layer, press the button 5. Die gewünschte Farbe kann entweder mit einem Klick auf die Zeichnen-Werkzeug-Schaltfläche 15 oder mit Hilfe des Menüs  Bearbeiten > Zeichnen-Werkzeug... > Auswahl Farbe  ausgewählt werden. Nach einem Doppelklick auf die Flächenebenen-Fläche lassen sich die Transparenz-Einstellungen sowie der Pinseldurchmesser (in Noppen) einstellen.

Die gesamte Fläche kann verschoben werden, indem gleichzeitig die STRG-Taste gedrückt, die Fläche mit der linken Maustaste angeklickt und die Maus zur gewünschten Position der Fläche bewegt wird. Hinweis: Enthält eine Flächenebene mehrere Flächen, so werden alle Flächen dieser Flächenenbenen verschoben.

Angemalte Flächen können gelöscht werden, indem der Pfeil neben der Zeichnungs-Werkzeug-Schaltfläche 15 angeklickt und das Löschen-Werkzeug ausgewählt wird. Unfortunately there's no way for now to copy/paste or move only a part of the area.

Textebenen

Mit Hilfe von Textebenen können verschienen Bereiche des Gleisplans beschriftet werden 14. Klicke auf Schaltfläche 6 und führe anschließend einen Doppelklick innerhalb des Ebenen-Fensterbereichs aus. Damit wird ein neues Fenter mit einem Textfeld geöffnet. Schreibe in diesem Textfeld den gewünschten Text. Die Schriftsatzeigenschaften wie z.B. Schriftgröße, Kursivschrift, etc. lassen sich einstellen. Klicke die Schaltfläche "ok" an, um das Textfeld-Fenster zu schließen. Schiebe anschließend mit der Maus das Textfeld auf die gewünschte Position. Zusätzlich kann der Text noch mit Hilfe der Drehungsschaltfläche und den Drehungseigenschaften gedreht werden. For more details about Text editing, please read the Using Texts section.

As for every type of layer, you can also change the transparency of the Text Layer by double clicking on the text layer tab 10. So if you want to have different transparencies for different texts, just use different Text Layers.

Ruler Layers

Ruler Layers are useful to measure distances on your layout. You can add a new Ruler Layer by pressing the button 7. In a Ruler Layer you can add linear and circular rulers which can display a distance in various unit (Stud, LDU, Straight Track, Module, Meter, Feet). These rulers can also be used to simply draw lines and circles on your layout, for example to delimit the wall of the room where you plan to build your layout. Rulers can also be attached to parts underneath such as the ruler will automatically extends and updates its measured value when you move the part. For more details about Ruler editing, please read the Using Rulers section.
