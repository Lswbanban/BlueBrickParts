The Part Library

In this chapter you can find how to use the Part Library. This includes a description of its organization and some display preferences and also how to filter it in order to find a specific part more easily.

But if you are looking for creating and adding some new parts to the Part Library, please read the How to extend the Part Library section. Alternatively, if you just want to group some existing parts and save your arrangement in the Library for later use, please read the How to save a group in the Library section. Later on if you want to share your new parts and contribute to the official Part Library please visit the Eurobricks forum here.
