Teile verschieben


Verschieben von Teilen ist einfach möglich: mit der Maus auf das zu verschiebene Teil zeigen, die linke Maustaste gedrückt halten und das Teil zur gewünschten Position verschieben. Die Rasterauflösung kann durch Anklicken des Pfeils neben der Schaltfläche Raster (10) und anschließender Auswahl der Auflösung aus der Auswahlliste geändert werden. Die Rasterauflösung ist in Noppen angegeben. Klickt man nicht auf den Pfeil neben der Schaltfläche Raster sondern direkt auf diese Schaltfläche, wird die Verbindung zum Raster aufgehoben und Teile lassen sich beliebig plazieren.
Soll eine Gruppe von Teilen, z.B. ein Gleisabschnitt, bewegt werden, so sollte man mit der Maus auf den Verbindungspunkt gehen, mit dem diese Gruppe an ein anderes Teil angebunden werden soll. Dann die linke Maustaste gedrückt halten und die Gruppe zur gewünschten Position verschieben. Andernfalls kann es sein, dass diese Gruppe nicht mit dem gewünschten Verbindungspunkt verbunden wird sondern am Raster ausgerichtet wird.

Drehen von Teilen ist mit der Schaltfläche (12) (entgegen Uhrzeigersinn) bzw. (13) (im Uhrzeigersinn) einfach durchzuführen. Durch Anklicken der Schaltfläche "Rotationsschritte" (11) und anschließender Auswahl der dort angegebenen Winkel kann festgelegt werden, um wieviel Grad das Teil beim Anklicken der Schaltflächen "Drehung" gedreht wird. Die Auswahl von 22.5° ist ideal um Gleiskurven zu verbinden. In den meisten Fällen ist die Einstellung auf 90° optimal. Wird ein neues Teil durch Anklicken dieses Teils in der Teileliste (Fensterausschnitt rechts oben) oder durch Benutzung von Tastenkürzel eingefügt, wird das Teil automatisch so gedreht, dass es zum Verbindungspunkt des davor ausgewählten bzw. angefügten Teiles passt. Wird ein bereits verbundenes Teil gedreht, so erfolgt die Drehung so dass die Verbindung erhalten bleibt, unabhängig davon welcher Drehwinkel mit der Schaltfläche "Rotationswinkel" (11) eingestellt wurde. Wird eine bereits vernüpfte Kurve gedreht, so wird aus einer Linkskurve eine Rechtskurve (und umgekehrt). Ist das letzte Teil eine Rechtskurve, werden alle weiteren, durch Anklicken in der Teileliste angefügten Kurven als Rechtskurven angefügt. Analoges gilt für Linkskurven. Alternativ zur Nutzung der Schaltflächen (12) und (13) kann ein bereits verbundenes Teil auch mit Hilfe der Leertaste gedreht werden.

Ausrichten von Teilen geschieht auf folgende Weise: Entweder wird das Teil mit einem, bereits am Raster ausgerichteten Teil verbunden oder das Teil wird an einer beliebigen Stelle abgelegt. Das Teil wird dann automatisch am Raster ausgerichtet. Die Rasterweite kann mit der Schaltfläche (10) eingestellt werden.

Verbinden von Teilen erfolgt automatisch, wenn Teile der gleichen Kategorie (Gleis, Straße, Duplo, etc.) nebeneinander gelegt werden. Die Verbindungspunkte von Teilen unterschiedlicher Kategorien haben unterschiedliche Farben, so sind z.B. Verbindungspunkte von Gleisen gelb und Verbindungspunkte von Straßen blau. Nur Teile der gleichen Kategorie können miteinander verbunden werden. Nach einer erfolgreichen Verbindung werden an der Verbindungsstelle die Verbindungspunkte nach dem Loslassen der linken Maustaste nicht mehr angezeigt. Wurden die Teile nicht verbunden, werden die Verbindungspunkte weiterhin angezeigt. Sollen Verbindungspunkte generell nicht angezeigt werden, so muss die Option "Zeige freie Verbindungspunkte an" im Menü "Bearbeiten", "Allgemeine Einstellungen", Karteireiter Aussehen, deaktiviert werden.
Hinweis: Durch eine Verbindung werden Teile nicht automatisch zueinander passend ausgerichtet. So kann z.B. ein waagerechtes Gleis direkt mit einem senkrechten Gleis verbunden werden. Die so entstandene 90° Ecke kann in der Realität nicht gebaut werden. Soll das anzufügende Teil automatisch passend ausgerichtet werden, so muss zunächst der gewünschte Verbindungspunkt markiert werden (zunächst das Teil anklicken, ggf. mit der Eingabetaste den gewünschten Verbindungspunkt auswählen). Anschließend muss das gewünschte Teil in der Teileliste mit der linken Maustaste angeklickt werden. Dieses Teil wird dann am aktuellen Verbindungspunkt angefügt und automatisch passend ausgerichtet.
Bei der Verbindung einer Gruppe von Teilen (z.B. ein Gleisabschnitt) sollte das zu verbindende Teil dieser Gruppe angeklickt und verschoben werden. Wird ein anderes Teil dieser Gruppe angeklickt und mit diesem Teil die Gruppe verschoben, kann es passieren, dass dieses Teil und nicht das gewünschte Teil mit einem anderen Teil des Gleisplans verbunden wird.

In den Vordergrund bzw. Hintergrund rücken ist eine Funktion, die besonders dann benötigt wird, wenn Teile aus unterschiedlichen Kategorien in der gleichen Ebene verbaut werden, z.B. Grundplatten und Gleise, oder wenn eine Brücke im Gleisplan enthalten ist und eindeutig sein soll, welches Gleis über und welches unterhalb der Brücke verläuft. Mit der Schaltfläche (14) und (15) lassen sich Teile in den Vordergrund bzw. in den Hintergrund rücken. Wie weit ein Teil im Vorder- bzw. Hintergrund plaziert ist, hängt davon ab, wann das Teil eingefügt wurde. Die zuerst eingefügten Teile sind am weitesten im Hintergrund, die zuletzt eingefügten Teile befinden sich im Vordergrund. Alternativ zur Benutzung der Schaltflächen (14) bzw. (15) kann auch die rechten Maustaste gedrückt werden. Durch Auswahl der im Auswahlmenü erscheinenden Optionen "In den Vordergrund" bzw. "In den Hintergrund" wird dann das Teil in den Vorder- bzw. in den Hintergrund geschoben.

Alle diese Aktionen können rückgängig gemacht und auch wieder hergestellt werden, für Details siehe Abschnitt Rückgängig machen und wiederherstellen.
